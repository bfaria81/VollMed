# Projeto 
Voll.med é uma clínica médica fictícia que necessida de um aplicativo para gestão de consultas. 

### Funcionalidades

- [x] CRUD de médicos;
- [x] CRUD de pacientes;
- [x] Agendamento de consultas;
- [x] Cancelamento de consultas.
